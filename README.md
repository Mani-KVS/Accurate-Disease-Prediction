# 🩺 Disease Prediction System

## 📌 Overview
This project predicts diseases based on user symptoms using machine learning.

## ⚙️ Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

## 🧠 Models Used
- Decision Tree
- Naive Bayes
- Random Forest
- Ensemble (Soft Voting)

## 📊 Features
- 132 symptom-based inputs
- High accuracy prediction
- Multiple model comparison
- Ensemble learning

## 📁 Files
- Untitled4.ipynb → main model
- Training.csv → training data
- Testing.csv → testing data

## 🚀 How to Run
1. Install dependencies
2. Open notebook
3. Run all cells

## 📈 Output
Predicts disease based on symptoms
